但是我就是喜欢用vim啊

vim是附着在noilinux终端里的一个文本编辑器，功能很是强大，但是一开始看起来却是个弱鸡

1，进入vim

noilinux桌面的左上角是应用程序，进入应用程序-附件，就可以找到终端了。

通过终端，我们可以操控电脑进行一些操作，一些是可以直接在GUI中完成的，一些是底层操作，只能在终端里用。vim是附着于终端的，你很难用其他方式打开它。

vim指令格式：

vim [你要打开/创建的文件]

在这里比如说按照我写的堆的模板来说的话，我就输入vim heap.cpp

接下来让我们进入神圣的vim

2，使用vim

vim有一个配置文件，默认情况下是空的。但是通过映射和其他一系列操作可以搞成一个极其优秀的IDE。但是在考场上，也没必要搞得那么好了。

当然在家里使用时就可以对自己好一点~

在终端输入vim ～/. vimrc，然后输入

set nu
set tabstop=4 
imap [ []<ESC>i 
imap ( ()<ESC>i 
imap { {}<ESC>i
imap " ""<ESC>i
imap ' ''<ESC>i

这样，手感会好很多。

进入vim的界面以后，输入I，你会发现vim的最底下变成了插入。接下来就可以开始输入了。注意vim没有自动缩进，想要设置的可以上网搜一下。

3，退出vim

退出vim

绝对不能直接关闭！

直接关闭，vim不会自动保存，所以你的文件就这么丢了。应该像这样：

[ESC][:][w][q]

ESC可以进入底部命令行

冒号代表这是一条命令

w是保存

q是退出

像这样，你的文件就被保存啦~

另外可以随时输入[ESC][:][w]来保存

4，编译运行

g++ [文件名]

编译完成以后，可以使用mv a.out [你想要的文件名].out来改名，然后使用./新文件名.out运行，这些都是常识了。

5，总结

vim功能强大，但是门槛略高，同时它也是一个非常有意思的编辑器。
